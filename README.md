# Rush Downward

**一款基于Cocos Creator开发的“掉落生存”类休闲小游戏。**

## 🎮 游戏简介
玩家控制角色在平台间不断向下跳跃。平台会持续上升，一旦触碰到上边界或碰到地刺游戏就会结束。
这是我的第一款独立开发游戏作品，是“是男人就下一百层”玩法的复刻与创新尝试。

## 🕹️ 核心玩法
*   **左右移动**：避开地刺，尽可能生存更久。
*   **血条系统**：触碰地刺会扣血。
*   **道具系统**：计划加入**护盾、回溯、无敌**等特色道具。
*   **无尽挑战**：生存越久，上升速度越快，挑战你的极限反应。

## 🛠️ 技术栈
*   **游戏引擎**：Cocos Creator
*   **开发语言**：JavaScript/TypeScript
*   **发布平台**：Web (HTML5)
