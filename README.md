## 🎯 核心玩法

- **左右移动**：点击屏幕左侧/右侧（或使用键盘 A/D）控制角色移动。
- **生存挑战**：平台会不断上升，碰到顶部边界或地刺即游戏结束。
- **血条系统**：触碰地刺会扣除血量，血条归零则失败。
- **道具系统（v1.0 已实现）**：护盾道具可抵挡一次伤害，回溯道具可回到三秒前的位置。
- **（计划中）** 无敌道具、无尽模式、星级评定。

## 🛠️ 技术栈与实现亮点

- **游戏引擎**：Cocos Creator
- **开发语言**：TypeScript / JavaScript
- **核心机制**：
  - 随机平台生成（保证每局体验不同）
  - 道具系统（护盾、回溯等）
  - 血条管理与游戏状态机
- **版本管理**：Git + GitHub，完整提交历史可追溯
